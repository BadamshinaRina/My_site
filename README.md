# Детский сад «Солнышко» ☀️  

Учебный проект — сайт детского сада с адаптивным дизайном, формой заявки и публикацией через GitHub Pages.

---

## 🔧 Используемые технологии
- HTML5  
- CSS3 (Flexbox, градиенты, анимации, псевдоэлементы)  
- Git / GitHub  
- GitHub Pages  

---

## 📁 Структура проекта
index.html — главная страница
contacts.html — контакты и адрес
form.html — форма заявки в садик
css/style.css — все стили
images/ — изображения
